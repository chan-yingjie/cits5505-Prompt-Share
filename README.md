## Application Purpose

This web application is designed as an interactive LLM Prompt Battle and Optimization Platform. The system allows users to submit prompts and compare their effectiveness through community interaction.

Users can view prompts submitted by others, explore generated outputs, vote on prompt quality, and provide feedback through comments. By comparing different prompts and observing their results, users can learn how prompt design influences LLM responses.

The platform aims to support collaborative learning by enabling users to refine prompts based on community feedback and rankings. It provides an engaging environment for experimenting with prompt engineering and understanding how to improve interactions with large language models.

Overall, the application combines user interaction, evaluation, and feedback to help users develop better prompt engineering skills.

## How to Launch the Application

- Clone or download the project repository to your local machine.

- Set up the required development environment, including installing the necessary runtime and dependencies.

- Navigate to the project directory on your computer.

- Start the application using the appropriate method for the project.

- Open a web browser and access the application through the local host address provided after launching.

- If required, register a new user account or log in with existing credentials.

- Use the application features as intended after successful login.

- If any issues occur, check that all dependencies are installed correctly and that the environment is properly configured.


## How to Run Tests

Currently, the application does not include automated test scripts.

To verify that the system is working correctly, users can perform manual testing by following these steps:

1. Register a new user account.
2. Log into the application.
3. Submit a prompt.
4. View the generated output or displayed response.
5. Vote on a prompt.
6. Add a comment to a prompt.
7. Browse prompts submitted by other users.

If all of the above actions work correctly, the core features of the application are functioning as expected.
